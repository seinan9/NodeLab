# NodeLab
NodeLab is a Python framework for research, enabling easy experimentation, flexibility, and reproducibility with a node-based approach.

## Installation

To install NodeLab, follow these steps:

1. Clone the repository:
   
  ```bash
  git clone https://github.com/seinan9/NodeLab.git
  ```

2. Navigate to the cloned directory:
   
  ```bash
  cd nodelab
  ```

3. Install the library using `pip`:

 ```bash
 pip install .
 ```
This will install NodeLab and it's dependencies (pyyaml).
